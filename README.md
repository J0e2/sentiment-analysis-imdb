# 🎭 Sentiment Analysis — IMDB Reviews

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white) ![TensorFlow](https://img.shields.io/badge/TensorFlow-FF6F00?style=flat-square&logo=tensorflow&logoColor=white) ![HuggingFace](https://img.shields.io/badge/HuggingFace-FFD21E?style=flat-square&logo=huggingface&logoColor=black) ![Jupyter](https://img.shields.io/badge/Jupyter-F37626?style=flat-square&logo=jupyter&logoColor=white)

> **Transformer-based** binary sentiment classifier on 50,000 IMDB movie reviews.

---

## 📋 Overview

Classifies IMDB movie reviews as Positive or Negative using a transformer encoder with attention mechanism. Fine-tuned on the full 50K IMDB dataset.

## 🎯 Results

| Metric | Value |
|--------|-------|
| Task | Binary Sentiment (Pos / Neg) |
| Dataset | IMDB 50,000 reviews |
| Architecture | Transformer + Attention |

## 🔧 Key Techniques

- ✅ Transformer encoder with multi-head self-attention
- ✅ Tokenization + padding/truncation
- ✅ Dropout regularization
- ✅ Binary cross-entropy loss

## 🚀 Run

```bash
pip install -r requirements.txt
jupyter notebook sentiment_analysis.ipynb
```
